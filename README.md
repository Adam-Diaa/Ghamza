Ghamza (meaning "wink" in Arabic) is a digital version of the classic party game where one player is secretly chosen as the "killer" who eliminates other players by winking at them, while the remaining players try to identify the killer before everyone is eliminated.
This mobile app eliminates the need for physical cards and makes the game setup quick and easy. Perfect for parties, family gatherings, and friend meetups!
